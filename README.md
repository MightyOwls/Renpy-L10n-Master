# 🚀 Ren'Py L10n Master (Ren'Py 汉化大师)

一款专为 Ren'Py 视觉小说打造的**工业级**自动化本地化工具。集成脱机解包、异步大模型并发、以及智能防爆翻译管线，将动辄数月的游戏汉化周期压缩至数小时。
目前处于项目初期阶段，但基本功能已经就绪。已成功通过Socially Awkward、Temptation Ballad、After Class、Dawn Chorus等游戏的中文汉化测试。之后仍会继续测试并发布新版本。
请尽量选择AI翻译功能，Google Translate虽然免费但非常不稳定。类似Deepseek之类的中文大模型比较适合游戏的翻译，价格比较便宜、亲民。

## ✨ 核心特性 (Features)

- **⚡ 极限异步并发**：底层采用 `aiohttp` 协程架构，全面压榨大模型 API 吞吐量。
- **🛡️ 纯血锚点通讯 (Zero-JSON)**：独创“行号索引锚定法”，彻底解决 LLM 翻译长文本时因嵌套引号或截断导致的 JSON 崩溃问题，容错率 100%。
- **📦 脱机解包引擎**：内嵌 `rpatool` 与 `unrpyc`，零依赖、断网环境下依然可实现 100% 还原游戏底层源码。
- **🩺 智能漏译狙击**：自带二次查漏补缺模块，精准定位网络波动造成的残留英文，避免全量重翻浪费 API 额度。
- **👁️ 视网膜级高分屏适配**：深度调用 Windows 底层 API，2K/4K 屏幕下界面依然锐利清晰。

## 🛠️ 如何使用 (Quick Start)

1. 在本仓库的 [Releases] 页面下载最新版的 `.exe` 独立运行包。
2. 选择你需要汉化的 Ren'Py 游戏根目录。
3. 填入你的 DeepSeek / OpenAI API Key。
4. 点击“启动全量汉化”，喝杯咖啡，等待奇迹发生。

## ⚠️ 保护声明
本工具在提取和翻译过程中，会通过正则掩码强行隔离并保护所有 Ren'Py 底层变量（如 `[player]`, `{color}` 等），绝不会破坏游戏的原始逻辑结构。
