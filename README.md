## 🛠️ 快速上手 (Quick Start)

1.  **定位基地：** 选择您的 Ren'Py 游戏根目录。
2.  **暴力部署：** 点击【⚙️ 暴力解包与提取】，解压剧本并生成 `tl/schinese` 翻译文件夹。
3.  **字体注入：** 点击【🔠 注入自带黑体】（覆盖全游戏 UI），并点击【🔤 注入中英菜单】。
4.  **极速估算：** 配置好您的 API Key 和并发数，点击【📊 极速估算 & 工作台】，评估翻译成本并设置特定术语（人名/地名）保护白名单。
5.  **启动全量汉化：** 冲杯咖啡，欣赏进度条丝滑推进的艺术！
